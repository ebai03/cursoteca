# Scripts de configucación Nexcloud

**NO USAR**

Este directorio contiene scripts para configurar Nexcloud en un servidor nuevo. Se utilizó como prueba de concepto pero el proyecto fue posteriormente migrado a otros servicios.

Nextcloud tiene bastantes funcionalidades pero por cuestión de mantenimiento y accesibilidad al público se decidió el cambio.

Aún así, estos scripts pueden ser útiles para configurar un servidor nuevo con Nexcloud.
