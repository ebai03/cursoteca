Estructura del repositorio
por año académico y ciclo

Los archivos están organizados en 5 niveles jerárquicos, desde lo general (año de carrera) hasta lo específico (material de un semestre concreto).

Estructura del repositorio

Estructura general del repositorio:

data/
├── Año_1
│   ├── Ciclo_1
│   │   ├── X_introduccion_a_la_computacion_CI0110
│   │   └── X_introduccion_a_la_matematica_para_computacion_MA0291
│   └── Ciclo_2
│       ├── X_calculo_para_computacion_i_MA0293
│       ├── X_estructuras_discretas_CI0111
│       └── X_programacion_i_CI0112
├── Año_2
│   ├── Ciclo_1
│   │   ├── X_algebra_lineal_para_computacion_MA0292
│   │   └── ... (otros cursos)
│   └── Ciclo_2
│       └── ... (cursos)
└── Año_3, Año_4, Optativas
    └── ... (estructura similar)

Nota: Las carpetas prefijadas con X_ son placeholders temporales sin contenido. Aparecen aquí para mostrar la estructura completa del plan de estudios.

Contenido dentro de una instancia de curso:

Ruta ejemplo: data/Año_1/Ciclo_2/Programacion_I_CI0112/2025_Semestre_I/ 

2025_Semestre_I/
├── Carta_al_Estudiante_CI0112_I_2025.pdf
├── 01_Material/
│   ├── Tema_01_Punteros.pdf
│   ├── Tema_02_Memoria_Dinamica.pdf
│   └── Tema_03_Estructuras.pdf
├── 02_Quices/
│   ├── Quiz_1_Enunciado.pdf
│   ├── Quiz_2_Enunciado.pdf
│   └── Quiz_3_Enunciado.pdf
└── 03_Examenes_Enunciados/
    ├── Parcial_1_Enunciado.pdf
    ├── Parcial_2_Enunciado.pdf
    └── Final_Enunciado.pdf
¿Cómo contribuir?

Si tenés material académico que querés compartir — o si encontraste información que debe modificarse o censurarse — podés contribuir directamente al repositorio.

01 · Pull Request

Agregar material

Creá un pull request en aecci/banco_examenes respetando la estructura del repositorio detallada en la sección anterior. Tus cambios serán revisados antes de publicarse.

02 · Modificar o censurar

Solicitar cambios

Si necesitás eliminar o censurar información personal en un documento existente, también podés usar el mismo flujo de pull request en aecci/banco_examenes.

03 · Archivos grandes

Contacto directo

Si un archivo es demasiado grande para añadirse a Git, contactá a un administrador directamente por correo electrónico: cursoteca@aecci-ucr.com

1 · Fork del repo → 2 · Agregás el material → 3 · Abrís un Pull Request → 4 · Se publica
